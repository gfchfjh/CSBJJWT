🚀 快速开始

Windows用户:
───────────
1. 解压安装包
2. 双击运行 install.bat (首次安装)
3. 双击运行 start_backend.bat (启动后端)
4. 双击运行 start_frontend.bat (启动前端)
5. 浏览器访问 http://localhost:5173

Linux/Mac用户:
──────────────
1. 解压安装包
2. chmod +x install.sh start_backend.sh start_frontend.sh
3. ./install.sh (首次安装)
4. ./start_backend.sh (启动后端)
5. ./start_frontend.sh (启动前端)
6. 浏览器访问 http://localhost:5173
