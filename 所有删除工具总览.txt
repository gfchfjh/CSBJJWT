🎯 推荐方案

方案 1：自动扫描删除（⭐推荐，最智能）
────────────────────────────────────────
📄 脚本：自动扫描删除KOOK.bat
📖 说明：使用自动扫描删除.txt

优点：
✓ 自动扫描整个系统
✓ 找到所有 KOOK 文件（无论在哪里）
✓ 深度清理（注册表、缓存、临时文件）
✓ 显示完整清单，安全可靠

适用场景：
• 不确定文件在哪里
• 曾经移动或复制过文件夹
• 想要彻底清理
• 其他脚本没删干净

使用方法：
1. 右键"以管理员身份运行" 自动扫描删除KOOK.bat
2. 等待扫描完成（30秒-2分钟）
3. 查看清单，输入 YES 确认
4. 等待删除完成（2-5分钟）


方案 2：标准删除（快速简单）
────────────────────────────────────────
📄 脚本：删除本地KOOK系统.bat
📖 说明：立即删除指南.txt

优点：
✓ 执行速度快
✓ 针对标准安装优化
✓ 界面友好，有彩色提示

适用场景：
• 知道文件在标准位置
• 使用一键安装脚本安装的
• 想要快速删除

使用方法：
1. 右键"以管理员身份运行" 删除本地KOOK系统.bat
2. 输入 YES 确认
3. 等待完成（2-5分钟）


方案 3：通用删除
────────────────────────────────────────
📄 脚本：KOOK_完全卸载.bat
📖 说明：如何完全删除.txt

优点：
✓ 兼容性好
✓ 适用于各种安装方式

适用场景：
• 通用删除场景
• 其他脚本无法运行时

使用方法：
1. 右键"以管理员身份运行" KOOK_完全卸载.bat
2. 输入 YES 确认
3. 等待完成（2-5分钟）


方案 4：仅清理开发文件（保留源码）
────────────────────────────────────────
📄 脚本：清理开发文件.bat
📖 说明：（脚本内有详细说明）

优点：
✓ 保留源代码
✓ 只删除构建产物和依赖
✓ 释放最多空间（~1.7 GB）

适用场景：
• 开发者
• 想保留源码但清理构建文件
• 释放磁盘空间

使用方法：
1. 双击运行 清理开发文件.bat（不需要管理员权限）
2. 输入 Y 确认
3. 等待完成

场景 1：不确定文件在哪里
──────────────────────────
推荐：自动扫描删除KOOK.bat ⭐⭐⭐⭐⭐

理由：
• 自动扫描所有位置
• 不会遗漏任何文件

场景 2：标准安装，知道位置
──────────────────────────
推荐：删除本地KOOK系统.bat ⭐⭐⭐⭐

• 针对标准安装优化

────────────────────────────

────────────────────────────

否 → 继续


问题 3：其他脚本删除失败了吗？
────────────────────────────
是 → 用"自动扫描"（自动扫描删除KOOK.bat）⭐
否 → 任选一个


问题 4：想要最彻底的清理吗？
────────────────────────────
是 → 用"自动扫描"（自动扫描删除KOOK.bat）⭐
否 → 用"标准删除"（删除本地KOOK系统.bat）


════════════════════════════════════════════════════════════════
📝 注意事项
════════════════════════════════════════════════════════════════

所有脚本共同注意事项：
──────────────────────────
1. 删除前备份配置（如需保留）
2. 关闭所有 KOOK 相关程序
3. 以管理员身份运行（除"清理开发文件"外）
4. 删除后无法恢复


自动扫描删除特别注意：
──────────────────────────
1. 扫描需要时间（30秒-2分钟）
2. 会扫描多个硬盘
3. 删除前会显示完整清单
4. 请仔细检查清单，避免误删


清理开发文件特别注意：
──────────────────────────
1. 只删除构建产物，保留源码
2. 不需要管理员权限
3. 删除后需要重新安装依赖才能构建
