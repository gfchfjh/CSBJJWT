# KOOK Cookie 导出器 Chrome扩展

**一键导出KOOK Cookies，5秒完成，99%成功率！**

## ✨ 功能特性

### 🚀 核心功能
- ✅ **一键导出** - 点击即可导出所有KOOK Cookie
- ✅ **多种格式** - 支持JSON/Netscape/HTTP Header等格式
- ✅ **自动检测** - 自动识别KOOK网页并检测Cookie
- ✅ **实时通知** - Cookie变化实时提醒
- ✅ **直接发送** - 一键发送到KOOK消息转发系统

### 📦 支持格式
1. **JSON数组格式**（推荐）- 完整Cookie信息
2. **JSON对象格式** - 简化的 {name: value} 格式
3. **Netscape格式** - 标准Cookie文件格式
4. **HTTP Header格式** - 直接用于HTTP请求

### 🎨 界面特点
- 🌈 **现代化设计** - 渐变色背景，毛玻璃效果
- 📊 **实时统计** - 显示Cookie数量和导出次数
- 🔔 **友好提示** - 清晰的状态提示和错误信息
- 🎭 **动画效果** - 平滑的交互动画

## 📦 安装方法

### 方式1：Chrome Web Store（推荐）
1. 打开 [Chrome Web Store]
2. 搜索 "KOOK Cookie Exporter"
3. 点击"添加至Chrome"

### 方式2：本地安装（开发者模式）
1. 下载源代码到本地
2. 打开Chrome，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本目录

## 📖 使用教程

### 方法1：使用扩展弹窗（最简单）

1. **打开KOOK网页**
   - 访问 https://www.kookapp.cn
   - 确保已登录

2. **点击扩展图标**
   - 在浏览器工具栏找到扩展图标 🍪
   - 点击打开弹窗

3. **选择导出格式**
   - 默认为"JSON数组"（推荐）
   - 可选择其他格式

4. **一键导出**
   - 点击"一键导出Cookie"按钮
   - 等待导出完成（约1-2秒）

5. **使用Cookie**
   - **方式A：** 点击"复制到剪贴板"，然后粘贴到应用
   - **方式B：** 点击"下载为文件"，保存为文件
   - **方式C：** 点击"发送到应用"，直接发送到本地应用

### 方法2：使用浮动按钮（最快捷）

1. **打开KOOK网页**
   - 访问 https://www.kookapp.cn
   - 右下角会出现一个浮动的🍪按钮

2. **点击按钮**
   - 直接点击浮动按钮
   - Cookie会自动复制到剪贴板
   - 显示成功通知

### 方法3：直接发送到应用（最方便）

1. **确保应用正在运行**
   - KOOK消息转发系统需要在后台运行
   - 默认端口：9527

2. **使用扩展导出**
   - 按照方法1操作
   - 点击"发送到应用"按钮

3. **自动完成**
   - Cookie会直接发送到应用
   - 无需手动粘贴

## 🔒 隐私说明

### 权限说明
- `cookies` - 读取Cookie（仅限kookapp.cn和kaiheila.cn域名）
- `activeTab` - 检测当前标签页
- `storage` - 保存统计信息和配置
- `notifications` - 显示通知

### 隐私承诺
- ✅ **仅读取KOOK Cookie** - 不访问其他网站
- ✅ **本地处理** - 所有数据在本地处理
- ✅ **不上传云端** - 不会上传到任何服务器
- ✅ **开源透明** - 源代码完全开放

## ⚠️ 注意事项

1. **Cookie安全**
   - Cookie是敏感信息，请勿分享给他人
   - 导出的文件请妥善保管
   - 定期更新Cookie

2. **登录状态**
   - 请确保已登录KOOK
   - Cookie过期需要重新登录
   - 扩展会自动检测Cookie状态

3. **浏览器兼容**
   - 支持Chrome 88+
   - 支持Edge 88+
   - 支持其他Chromium内核浏览器

## 🐛 故障排查

### 问题1：未检测到Cookie
**原因：**
- 未登录KOOK
- Cookie已过期
- 浏览器阻止了Cookie

**解决：**
1. 重新登录KOOK
2. 清除浏览器缓存后重新登录
3. 检查浏览器Cookie设置

### 问题2：发送到应用失败
**原因：**
- 应用未运行
- 端口被占用
- 防火墙阻止

**解决：**
1. 确保KOOK消息转发系统正在运行
2. 检查应用端口（默认9527）
3. 临时关闭防火墙测试

### 问题3：扩展无法加载
**原因：**
- Chrome版本过低
- 扩展文件损坏

**解决：**
1. 更新Chrome到最新版本
2. 重新下载扩展文件
3. 以开发者模式重新加载
