下载: KOOK-Forwarder-4.1.0-Setup.exe

#### 从源码运行
```bash
# 克隆仓库
git clone https://github.com/gfchfjh/CSBJJWT.git
cd CSBJJWT

# 切换到v4.1.0分支（如果有）
git checkout v4.1.0

# 安装依赖
cd backend && pip install -r requirements.txt
cd ../frontend && npm install

# 启动服务
npm run dev
```
