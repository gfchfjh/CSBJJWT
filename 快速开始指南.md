## 📝 配置步骤

### 步骤1: 登录KOOK账号

**方式A：Cookie导入（推荐）**

1. 浏览器登录 https://www.kookapp.cn/
2. 按 `F12` 打开开发者工具
3. 切换到 `Console`（控制台）
4. 粘贴以下代码并回车：

```javascript
copy(JSON.stringify(document.cookie.split('; ').map(c => {
  const [name, value] = c.split('=');
  return {
    name: name,
    value: value,
    domain: '.kookapp.cn',
    path: '/',
    expires: -1,
    httpOnly: false,
    secure: true,
    sameSite: 'None'
  };
}), null, 2));
console.log('Cookie已复制到剪贴板！');
```

5. 在应用中粘贴Cookie并添加

**方式B：账号密码（简单）**

直接输入KOOK邮箱和密码

**详细教程**: [Cookie获取详细教程](docs/Cookie获取详细教程.md)

---

### 步骤2: 配置机器人

#### Discord

1. 进入目标Discord服务器
2. 服务器设置 → 集成 → Webhooks
3. 创建新Webhook，复制URL
4. 在应用中粘贴并测试

**教程**: [Discord配置教程](docs/Discord配置教程.md)

#### Telegram

1. 与 [@BotFather](https://t.me/BotFather) 对话
2. 发送 `/newbot` 创建Bot
3. 获取Bot Token
4. 将Bot添加到群组
5. 使用应用内工具获取Chat ID

**教程**: [Telegram配置教程](docs/Telegram配置教程.md)

#### 飞书

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 获取App ID和App Secret
4. 将机器人添加到群组

**教程**: [飞书配置教程](docs/飞书配置教程.md)

---

### 步骤3: 设置频道映射

1. 进入"频道映射"页面
2. 点击"智能映射"（自动匹配同名频道）
3. 或手动添加映射
4. 点击"测试消息"验证

---

### 步骤4: 启动服务

1. 点击右上角"启动服务"按钮
2. 查看"实时日志"确认消息转发
3. 完成！🎉

## 💡 常见问题

### Q: Cookie导入失败？

**A**: 
1. 确保已在浏览器登录KOOK
2. 复制的Cookie要完整
3. 参考 [Cookie获取详细教程](docs/Cookie获取详细教程.md)

### Q: 消息没有转发？

**A**: 检查清单
- [ ] KOOK账号状态是否在线？
- [ ] 机器人配置是否正确？
- [ ] 频道映射是否已设置？
- [ ] 服务是否已启动？
- [ ] 查看"实时日志"页面的错误信息

## ⚡ 快捷命令

### 开发相关

```bash
# 启动开发服务器
cd frontend && npm run dev

# 启动后端
```
