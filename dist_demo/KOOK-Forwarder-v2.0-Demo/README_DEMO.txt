🚀 快速开始
-----------

Windows用户:
1. 双击运行 start.bat

Linux/Mac用户:
1. 运行 ./start.sh

⚙️  完整部署
------------

后端:
1. cd backend
2. pip install -r requirements.txt
3. python -m app.main

前端:
1. cd frontend
2. npm install
3. npm run dev
