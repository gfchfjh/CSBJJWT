Q: 安装依赖失败？
A: 使用国内镜像:
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   npm install --registry=https://registry.npmmirror.com

Q: 端口被占用？
A: Windows: netstat -ano | findstr :9527 然后 taskkill /PID <PID> /F
   Linux: lsof -i :9527 然后 kill -9 <PID>

Q: 浏览器访问不了？
A: 检查:
   1. 后端是否启动成功？
   2. 前端是否启动成功？
   3. 防火墙是否允许？
   4. 访问 http://localhost:5173 而非 127.0.0.1

Q: 消息转发失败？
A: 检查:
   1. Bot配置是否正确（Webhook URL/Token）
   2. 网络连接是否正常
   3. 查看实时日志中的错误信息
   4. 检查目标平台API限流

更多问题查看: INSTALLATION_GUIDE.md
