## 🚀 快速开始流程

```
第1步：查看必要软件
├─ Python 3.11+  → https://www.python.org/downloads/
├─ Node.js 18+   → https://nodejs.org/
└─ Git          → https://git-scm.com/download/win
      ↓
第2步：安装软件（勾选 Add to PATH）
      ↓
第3步：重启电脑
      ↓
第4步：运行 一键安装命令.bat（右键以管理员身份）
      ↓
第5步：等待完成（15-20分钟）
      ↓
第6步：双击 启动KOOK系统.bat
      ↓
完成！浏览器自动打开
```
