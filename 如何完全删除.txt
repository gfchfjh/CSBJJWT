================================================================
KOOK 系统 - 完全删除指南
================================================================

更新日期: 2025-11-03

================================================================
方法一：一键卸载（推荐，最简单）
================================================================

1. 找到文件 "KOOK_完全卸载.bat"

2. 右键点击 → "以管理员身份运行"

3. 输入 YES 确认

4. 等待完成（约 2-5 分钟）

5. 完成！所有文件已删除


================================================================
方法二：手动删除（如果一键卸载失败）
================================================================

步骤 1: 卸载应用程序
--------------------
1) 按 Win + I 打开设置
2) 点击 "应用" → "应用和功能"
3) 搜索 "KOOK"
4) 点击卸载


步骤 2: 删除用户数据
--------------------
1) 按 Win + R，输入以下内容并回车，然后删除文件夹：

   %APPDATA%\KOOK消息转发系统
   %LOCALAPPDATA%\KOOK消息转发系统
   %LOCALAPPDATA%\Programs\KOOK消息转发系统


步骤 3: 删除项目文件
--------------------
如果你有下载或克隆的源码，删除以下位置：

   C:\Users\你的用户名\KOOK-Build\
   C:\Users\你的用户名\Desktop\CSBJJWT\


步骤 4: 删除快捷方式
--------------------
删除桌面上的 "KOOK消息转发系统" 快捷方式


步骤 5: 清理临时文件
--------------------
1) 按 Win + R，输入 cleanmgr 回车
2) 选择驱动器，勾选"临时文件"
3) 点击确定


================================================================
方法三：只删除开发文件（保留配置）
================================================================

如果你是开发者，只想清理开发文件释放空间：

1. 找到文件 "清理开发文件.bat"
2. 双击运行
3. 输入 Y 确认
4. 等待完成

这将删除：
  ✓ backend/venv/ (约1GB)
  ✓ frontend/node_modules/ (约500MB)
  ✓ 构建产物
  ✓ 缓存文件


================================================================
验证是否删除干净
================================================================

方法1: 搜索文件
--------------
1) 打开资源管理器
2) 搜索 "KOOK"
3) 如果没有结果，说明已删除干净


方法2: 检查常用位置
-------------------
1) 打开 %APPDATA%
   → 不应该有 "KOOK消息转发系统" 文件夹

2) 打开 %LOCALAPPDATA%
   → 不应该有 "KOOK消息转发系统" 文件夹

3) 查看桌面
   → 不应该有 "KOOK消息转发系统" 快捷方式


================================================================
删除后如何重新安装？
================================================================

如果以后想重新安装：

1. 访问 GitHub: https://github.com/gfchfjh/CSBJJWT

2. 下载最新版本

3. 按照安装指南操作


================================================================
常见问题
================================================================

Q: 删除后会影响其他程序吗？
A: 不会。只删除 KOOK 系统相关文件。

Q: 会删除 Python 和 Node.js 吗？
A: 不会。Python 和 Node.js 保持不变。

Q: 如何备份配置？
A: 在删除前，复制这个文件到其他位置：
   %APPDATA%\KOOK消息转发系统\config.json

Q: 删除后还能恢复吗？
A: 如果没有备份，无法恢复。
   如果有备份，重新安装后恢复配置文件即可。
