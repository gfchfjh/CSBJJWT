## 💡 方案A：Web 版本（最推荐）

**总耗时**：10分钟

### 完整命令（复制粘贴即可）

```bash
# 1. 下载项目
git clone https://github.com/gfchfjh/CSBJJWT.git
cd CSBJJWT

# 2. 安装后端
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
cd ..

# 3. 安装前端
cd frontend
npm install --legacy-peer-deps
npm run build
cd ..

# 4. 启动（创建启动脚本）
```

**创建 `quick_start.bat` 文件，内容如下**：

```batch
@echo off
title KOOK 消息转发系统

cd backend
call venv\Scripts\activate.bat

echo 正在启动后端服务...
echo 请稍等 10 秒...
echo.

start cmd /k "python -m app.main"
timeout /t 10 /nobreak

echo 在浏览器中打开: http://127.0.0.1:9527
start http://127.0.0.1:9527

echo.
echo 系统已启动！关闭命令行窗口将停止服务。
pause
```

**使用**：
1. 双击 `quick_start.bat`
2. 等待浏览器自动打开
3. 开始使用！

## 🖥️ 方案C：开发模式（推荐技术用户）

**总耗时**：20分钟

### 完整命令

```bash
# 1-3 步同方案A

# 4. 启动开发模式
```

**创建 `start_dev.bat` 文件**：

```batch
@echo off
echo 启动开发模式...

start "后端" cmd /k "cd backend && call venv\Scripts\activate.bat && python -m app.main"
timeout /t 15 /nobreak
start "前端" cmd /k "cd frontend && npm run electron:dev"

echo 启动完成！
```

**使用**：
- 双击 `start_dev.bat`
- 会打开桌面应用窗口

## 🐛 遇到问题？

### 常见问题快速修复

**端口被占用**：
```bash
netstat -ano | findstr :9527
taskkill /F /PID <进程ID>
```

**依赖安装失败**：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**权限不足**：
- 右键以管理员身份运行命令行

**杀毒软件拦截**：
- 添加项目文件夹到白名单
