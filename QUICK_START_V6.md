## 📥 第一步：下载和安装（2-3分钟）

### Windows

1. **下载安装包**
```
https://github.com/gfchfjh/CSBJJWT/releases/download/v6.0.0/KOOK-Forwarder-6.0.0-Setup.exe
文件大小: ~150MB
```

2. **运行安装程序**
   - 双击 `KOOK-Forwarder-6.0.0-Setup.exe`
   - 如果Windows Defender提示"未知发布者"，点击"仍要运行"
   - 选择安装路径（默认即可）
   - 点击"安装"
   - 等待2-3分钟

3. **自动启动**
   - 安装完成后自动启动
   - 或从开始菜单找到"KOOK消息转发系统"

### macOS

1. **下载DMG**
```
# Intel Mac
https://github.com/gfchfjh/CSBJJWT/releases/download/v6.0.0/KOOK-Forwarder-6.0.0-macOS-x64.dmg

# Apple Silicon (M1/M2/M3)
https://github.com/gfchfjh/CSBJJWT/releases/download/v6.0.0/KOOK-Forwarder-6.0.0-macOS-arm64.dmg
```

2. **安装应用**
   - 打开下载的DMG文件
   - 拖动"KOOK消息转发系统"到"Applications"文件夹
   - 等待复制完成

3. **首次启动**
   - 右键点击应用 → 选择"打开"（绕过安全检查）
   - 或使用命令: `xattr -cr "/Applications/KOOK消息转发系统.app"`

### Linux

1. **下载AppImage**
```bash
wget https://github.com/gfchfjh/CSBJJWT/releases/download/v6.0.0/KOOK-Forwarder-6.0.0-x64.AppImage
```

2. **添加执行权限**
```bash
chmod +x KOOK-Forwarder-6.0.0-x64.AppImage
```

3. **运行**
```bash
./KOOK-Forwarder-6.0.0-x64.AppImage
```

**提示**: 如果缺少依赖，运行:
```bash
# Ubuntu/Debian
sudo apt-get install libgtk-3-0 libnotify4 libnss3 libxss1 libxtst6

# Fedora/RedHat
sudo dnf install gtk3 libnotify nss libXScrnSaver libXtst
```

### 步骤2: 登录KOOK（1-2分钟）

#### 方式A: 使用Chrome扩展（推荐，仅需5秒）

1. **安装Chrome扩展**
   - 点击"安装Chrome扩展"按钮
   - 按照提示加载扩展（`chrome-extension/`文件夹）

2. **导出Cookie**
   - 在Chrome打开 https://www.kookapp.cn/ 并登录
   - 点击浏览器工具栏的扩展图标 🍪
   - 点击"导出Cookie到剪贴板"
   - 显示"✅ 成功导出 XX 个Cookie"

3. **导入到软件**
   - 回到配置向导
   - 在Cookie输入框按 Ctrl+V 粘贴
   - 点击"验证并继续"
   - 显示"✅ Cookie验证通过"

#### 方式B: 账号密码登录

1. 输入KOOK邮箱
2. 输入密码
3. 点击"登录并继续"
4. 如需验证码，按提示输入

#### 方式C: 手动导出Cookie

参考教程: `docs/Cookie获取详细教程.md`

### 步骤4: 配置Bot（1-2分钟）

选择要使用的平台（可多选）：

#### Discord配置（30秒）

1. **获取Webhook URL**
   - 在Discord服务器中，进入频道设置
   - 整合 → Webhook → 新建Webhook
   - 复制Webhook URL

2. **填入软件**
   - Webhook名称：`游戏公告Bot`（备注用）
   - Webhook URL：粘贴刚复制的URL
   - 点击"🧪 测试连接"
   - 显示"✅ 测试成功！"

#### Telegram配置（1分钟）

1. **创建Bot**
   - 在Telegram与 @BotFather 对话
   - 发送 `/newbot`
   - 按提示设置Bot名称
   - 复制Bot Token

2. **获取Chat ID**
   - 将Bot添加到目标群组
   - 点击软件中的"🔍 自动获取"
   - 自动检测Chat ID

3. **填入软件**
   - Bot Token：粘贴
   - Chat ID：自动填入或手动输入
   - 点击"🧪 测试连接"

#### 飞书配置（2分钟）

1. **创建自建应用**
   - 访问 https://open.feishu.cn/
   - 创建企业自建应用
   - 获取App ID和App Secret

2. **配置权限**
   - 开启"机器人"能力
   - 添加权限：发送消息、上传图片

3. **填入软件**
   - App ID：粘贴
   - App Secret：粘贴
   - 点击"🧪 测试连接"

**提示**: 可以先配置一个，后续随时添加其他平台

## ❓ 常见问题

### Q1: Cookie导入失败怎么办？

**推荐方案**: 使用Chrome扩展
1. 安装扩展（软件内有教程）
2. 在KOOK网页版导出
3. 粘贴到软件

**手动方案**: 
- 查看详细错误提示
- 按照建议修改格式
- 重新尝试

### Q2: 消息没有转发？

**检查清单**:
- [ ] 账号状态是否在线（绿色）
- [ ] Bot配置是否正确（测试连接）
- [ ] 频道映射是否已启用
- [ ] 消息是否被过滤规则拦截

**解决方法**:
1. 查看实时日志（📋 日志页面）
2. 查找失败原因
3. 根据错误提示修复

### Q3: 图片转发失败？

**可能原因**:
- 图片被防盗链（已自动处理）
- 图片过大（已自动压缩）
- 目标平台限制

**解决方法**:
1. 检查图片策略（设置 → 图片处理）
2. 推荐使用"智能模式"（默认）
3. 查看图片处理统计

### Q4: 如何停止转发？

**方法**:
- 主页 → 点击"⏸️ 停止服务"
- 或关闭窗口（最小化到托盘）
- 或完全退出应用

### Q5: 如何卸载？

**Windows**: 控制面板 → 程序 → 卸载
**macOS**: 删除Applications中的应用
**Linux**: 直接删除AppImage文件

**注意**: 配置数据保留在 `~/Documents/KookForwarder/`，可手动删除
