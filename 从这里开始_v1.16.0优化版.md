## 🚀 快速开始（3分钟）

### 步骤1: 安装浏览器扩展（仅需一次）

```bash
# Chrome浏览器
1. 访问 chrome://extensions/
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择: /workspace/chrome-extension/
```

### 步骤2: 启动应用

```bash
cd /workspace
./start.sh  # 或 start.bat (Windows)
```

### 步骤3: 完成配置

```
访问: http://localhost:5173/wizard

步骤1: 同意免责声明（30秒）
步骤2: 使用扩展导入Cookie（30秒）

完成！
```
