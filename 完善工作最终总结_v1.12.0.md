### Docker配置对比

```yaml

    build: .
    # 基础配置
```

**v1.12.0**: 三套环境

```yaml
# docker-compose.yml (基础)
services:
  redis:  # 独立Redis
  backend:  # 后端服务

# docker-compose.dev.yml (开发)
services:
  backend:
    volumes:
      - ./backend/app:/app/app  # 热重载
  redis-commander:  # Redis管理工具
  filebrowser:      # 文件浏览器

# docker-compose.prod.yml (生产)
services:
  redis:
    command: --appendonly yes  # 持久化
  backend:
    deploy:
      update_config:  # 滚动更新
  nginx:  # 反向代理
```

**提升**: 灵活的容器化部署方案
