## 🚀 第一步：启动应用

### Windows 用户

1. 双击桌面图标或开始菜单中的「KOOK消息转发系统」
2. 首次启动会弹出配置向导

### macOS 用户

1. 在「启动台」或「应用程序」中找到应用
2. 双击启动

### Linux 用户

1. 双击 `.AppImage` 文件
2. 或命令行运行：`./KOOK-Forwarder-*.AppImage`

### 步骤2：登录KOOK账号

![登录KOOK](../images/wizard-step2.png)

#### 方式A：Chrome扩展（推荐，5秒完成）🔥

1. **如果未安装扩展**：
   - 点击「下载Chrome扩展」
   - 解压zip文件
   - 打开Chrome，访问 `chrome://extensions/`
   - 开启「开发者模式」
   - 点击「加载已解压的扩展程序」，选择解压后的文件夹

2. **如果已安装扩展**：
   - 打开 [KOOK网页版](https://www.kookapp.cn) 并登录
   - 点击浏览器工具栏的扩展图标
   - 点击「导出Cookie」
   - 回到本应用，粘贴Cookie（Ctrl+V）
   - 点击「验证并添加」

#### 方式B：手动粘贴Cookie

1. 打开 [KOOK网页版](https://www.kookapp.cn) 并登录
2. 按 `F12` 打开开发者工具
3. 切换到「Application」标签
4. 左侧选择「Cookies」→「https://www.kookapp.cn」
5. 复制所有Cookie（或只复制重要的几个）
6. 粘贴到本应用

#### 方式C：账号密码登录

1. 输入KOOK邮箱和密码
2. 点击「登录」
3. 如需验证码，会弹出输入框

💡 **提示**：Cookie方式最稳定，推荐使用

---

### 步骤3：选择监听的服务器和频道

![选择服务器](../images/wizard-step3.png)

1. 应用会自动加载您的所有KOOK服务器
2. 勾选您想监听的服务器
3. 展开服务器，勾选具体频道
4. 点击「下一步」

💡 **建议**：先选择1-2个测试频道，成功后再添加更多

---

### 步骤4：配置转发目标（Bot）

![配置Bot](../images/wizard-step4.png)

#### 配置Discord Webhook

1. 在Discord服务器中，右键点击目标频道
2. 选择「编辑频道」→「整合」→「Webhook」
3. 点击「新建Webhook」
4. 复制Webhook URL
5. 粘贴到本应用，点击「测试连接」

📺 [查看详细视频教程](https://example.com/discord-tutorial)

#### 配置Telegram Bot

1. 在Telegram中与 `@BotFather` 对话
2. 发送 `/newbot`，按提示创建Bot
3. 复制Bot Token
4. 将Bot添加到目标群组
5. 在本应用中填入Token，点击「自动获取Chat ID」

📺 [查看详细视频教程](https://example.com/telegram-tutorial)

#### 配置飞书Bot

1. 访问 [飞书开放平台](https://open.feishu.cn)
2. 创建企业自建应用
3. 开启「机器人」能力
4. 复制App ID和App Secret
5. 将机器人添加到群组
6. 在本应用中填入配置

📺 [查看详细视频教程](https://example.com/feishu-tutorial)

---

### 步骤5：设置频道映射

![频道映射](../images/wizard-step5.png)

#### 方式A：一键智能映射（推荐）🎯

1. 点击「智能映射」按钮
2. 应用会自动匹配同名或相似的频道
3. 检查匹配结果，确认无误后点击「应用」

**匹配示例**：
- KOOK「#公告」 → Discord「#announcements」✅
- KOOK「#活动」 → Telegram「活动群」✅
- KOOK「#更新日志」 → 飞书「更新通知群」✅

准确率：优秀

#### 方式B：手动配置

1. 点击「添加映射」
2. 选择KOOK频道
3. 选择目标平台和Bot
4. 输入目标频道ID
5. 保存

## 💡 使用技巧

### 技巧1：一个频道转发到多个目标

KOOK的同一个频道可以同时转发到：
- Discord频道A
- Telegram群组B
- 飞书群组C

只需为同一个KOOK频道创建多个映射即可。

### 技巧2：使用过滤规则

**示例1：只转发官方公告**
```
过滤规则 → 关键词白名单 → 添加「官方」「公告」
```

**示例2：屏蔽广告**
```
过滤规则 → 关键词黑名单 → 添加「广告」「代练」「外挂」
```

### 技巧3：最小化到托盘

```
设置 → 服务控制 → 开启「最小化到托盘」
```

关闭窗口时，程序会最小化到系统托盘，不会退出。

## ❓ 常见问题

### Q1: 账号显示离线？

**可能原因**：
- Cookie过期
- 网络问题
- KOOK账号被封

**解决方法**：
1. 重新登录获取新Cookie
2. 检查网络连接
3. 确认账号状态正常

---

### Q2: 消息转发延迟很大？

**可能原因**：
- 队列积压
- 目标平台限流
- 网络不稳定

**解决方法**：
1. 查看队列大小
2. 减少映射数量
3. 检查网络速度

---

### Q3: 图片转发失败？

**可能原因**：
- 图片防盗链
- 图片过大
- 目标平台限制

**解决方法**：
1. 切换到「智能模式」（自动处理）
2. 调整图片压缩质量
3. 查看错误日志

---

### Q4: 如何卸载？

**Windows**：
```
控制面板 → 程序 → 卸载 KOOK消息转发系统
```

**macOS**：
```
应用程序 → 删除应用
```

**清理数据**：
```
手动删除：用户文档/KookForwarder 文件夹
```
