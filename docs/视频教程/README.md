## 🎥 录制规范

### 视频质量要求

- **分辨率**: 1920x1080 (1080p)
- **帧率**: 30fps
- **格式**: MP4 (H.264编码)
- **比特率**: 5-8 Mbps
- **音频**: AAC 128kbps

### 录制工具推荐

- **Windows**: OBS Studio / Camtasia
- **macOS**: QuickTime / ScreenFlow
- **Linux**: OBS Studio / SimpleScreenRecorder

### 内容规范

1. **开场**
   - 显示教程标题
   - 简短介绍（5-10秒）

2. **主体内容**
   - 清晰的步骤说明
   - 必要时添加字幕
   - 关键点高亮标注

3. **结尾**
   - 总结要点
   - 相关资源链接
   - 感谢观看

### 音频规范

- 清晰的人声解说
- 背景音乐音量控制（≤20%）
- 无杂音和回声
