验证是否删除干净
════════════════════════════════════════════════════════════════

方法 1：搜索文件
---------------
1. 打开"此电脑"
2. 在右上角搜索框输入：KOOK
3. 如果没有结果 → 已删除干净 ✓


方法 2：检查常用位置
-------------------
按 Win + R，输入以下内容并回车，看是否还有文件夹：

1. %APPDATA%
   → 不应该有"KOOK消息转发系统"文件夹

2. %LOCALAPPDATA%
   → 不应该有"KOOK消息转发系统"文件夹

3. 检查桌面
   → 不应该有"KOOK消息转发系统"快捷方式
   → 不应该有"CSBJJWT"文件夹


════════════════════════════════════════════════════════════════
如果脚本运行失败
════════════════════════════════════════════════════════════════

原因 1：没有管理员权限
解决：确保右键选择"以管理员身份运行"

原因 2：文件被占用
解决：
   1. 按 Ctrl + Shift + Esc 打开任务管理器
   2. 找到所有 KOOK 相关进程
   3. 右键 → 结束任务
   4. 重新运行删除脚本

原因 3：部分文件无法删除
解决：
   1. 重启电脑
   2. 重新运行删除脚本

手动删除（如果脚本完全失败）
════════════════════════════════════════════════════════════════

步骤 1：卸载应用
----------------
Win + I → 应用 → 应用和功能 → 搜索"KOOK" → 卸载


步骤 2：删除文件夹
------------------
按 Win + R，输入以下路径并删除：

%APPDATA%\KOOK消息转发系统
%LOCALAPPDATA%\KOOK消息转发系统
%LOCALAPPDATA%\Programs\KOOK消息转发系统


步骤 3：删除源码
----------------
删除以下位置的 CSBJJWT 或 KOOK-Build 文件夹：

- 桌面
- C:\Users\你的用户名\KOOK-Build
- C:\Users\你的用户名\Documents
- C:\Users\你的用户名\Downloads


步骤 4：删除快捷方式
--------------------
删除桌面上的"KOOK消息转发系统"快捷方式

常见问题
════════════════════════════════════════════════════════════════

Q：会删除 Python 和 Node.js 吗？
A：不会。只删除 KOOK 系统的文件。

Q：我的配置能恢复吗？
A：不能。删除后无法恢复，请提前备份。

Q：如何备份配置？
A：删除前，复制这个文件到其他位置：
   %APPDATA%\KOOK消息转发系统\config.json

Q：删除后还能重新安装吗？
A：可以。访问 GitHub 重新下载安装。

Q：会影响其他软件吗？
A：不会。只删除 KOOK 相关文件。
