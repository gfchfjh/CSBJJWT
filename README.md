## 📦 项目简介

KOOK消息转发系统是一款功能强大、易于使用的跨平台消息转发工具，能够将KOOK（原开黑啦）平台的消息实时转发到Discord、Telegram、飞书等其他平台。

### ✨ 核心特性

**易用性（v2.0.0 新增）：**
- 🎯 **一键安装** - 5分钟完成安装和配置
- 🔍 **智能检查** - 8项环境检查+自动修复
- 🧙 **配置向导** - 4步完成所有配置
- 📚 **完整帮助** - 内置教程+FAQ+视频

**智能化（v2.0.0 新增）：**
- 🎯 **智能映射** - 75%+准确率的频道匹配
- 🍪 **智能导入** - Cookie多种方式导入
- 🔬 **智能诊断** - 自动识别并修复问题
- 🛡️ **智能过滤** - 黑白名单+正则表达式

**性能优化：**
- ⚡ **高性能处理** - 消息处理速度提升3倍
- 🚀 **WebSocket** - 实时通信，CPU↓60%
- 💻 **虚拟滚动** - 10000+日志流畅显示
- 📊 **批量处理** - 数据库批量操作

**安全与稳定：**
- 🔒 **API Token** - 强制身份认证
- 🛡️ **速率限制** - 防止滥用攻击
- 🔐 **密码验证** - 强密码策略
- 📝 **审计日志** - 完整操作记录

**基础特性：**
- 🤖 **多平台支持** - Discord、Telegram、飞书
- 🔄 **实时转发** - 平均延迟<2秒
- 🎨 **格式转换** - 自动转换消息格式
- 🔗 **链接预览** - 自动提取标题、描述、图片
- 💾 **消息队列** - 保证不丢失任何消息
- 🌙 **深色主题** - 浅色/深色/自动三种模式
- 🌍 **国际化** - 中英文双语支持

### 方式1: 预编译安装包（⭐推荐）

```bash
# 1. 下载对应平台的安装包
# Windows: KOOK-Forwarder-2.0.0-win.exe
# macOS: KOOK-Forwarder-2.0.0.dmg
# Linux: KOOK-Forwarder-2.0.0.AppImage

# 2. 安装运行
# Windows: 双击运行，按向导安装
# macOS: 拖拽到应用程序文件夹
# Linux: chmod +x *.AppImage && ./KOO*.AppImage

# 3. 完成5步配置向导
# - 环境检查
# - Cookie导入
# - 频道配置
# - 转发测试
# - 完成启动
```

✅ 预计时间: 5分钟  
✅ 优点: 一键安装、零配置、开箱即用
