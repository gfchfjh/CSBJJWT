### 2. KOOK_完全卸载.bat

**类型**: 自动化脚本  
**适合**: 想要快速彻底卸载的用户  
**功能**:
- ✅ 一键卸载 Electron 应用
- ✅ 自动删除用户数据
- ✅ 清理构建目录
- ✅ 删除快捷方式
- ✅ 清理临时文件和缓存
- ✅ 清理注册表
- ✅ 验证清理结果

**使用方法**:
```bash
# 右键以管理员身份运行
KOOK_完全卸载.bat

# 输入 YES 确认
# 等待完成（2-5 分钟）
```

**删除内容**:
- Electron 应用 (~100 MB)
- 用户配置和数据 (~10 MB)
- 构建目录 KOOK-Build/ (~500 MB)
- 桌面和开始菜单快捷方式
- 临时文件和缓存 (~50 MB)
- 注册表项

**总计释放**: ~700 MB (不含源码和虚拟环境)

---

### 3. 清理开发文件.bat

**类型**: 开发者工具  
**适合**: 开发者，想保留源码但清理构建产物  
**功能**:
- ✅ 删除 Python 虚拟环境 (backend/venv/)
- ✅ 删除 Node.js 依赖 (node_modules/)
- ✅ 删除构建产物 (dist/)
- ✅ 清理缓存文件 (__pycache__, .vite)
- ✅ 保留源代码

**使用方法**:
```bash
# 在项目根目录运行
清理开发文件.bat

# 输入 Y 确认
```

**删除内容**:
- backend/venv/ (~1 GB)
- frontend/node_modules/ (~500 MB)
- dist/ (~200 MB)
- build/pyinstaller/ (~50 MB)
- Python 缓存 (~10 MB)

**总计释放**: ~1.7 GB

**不删除**:
- ✅ 源代码 (.py, .vue, .js 文件)
- ✅ 配置文件 (config.json)
- ✅ 文档 (.md 文件)

## ⚠️ 重要提示

### 卸载前请注意

1. **备份配置**（如果需要保留）:
   ```bash
   # 备份这些文件：
   %APPDATA%\KOOK消息转发系统\config.json
   %APPDATA%\KOOK消息转发系统\cookies.json
   %APPDATA%\KOOK消息转发系统\*.db
   ```

2. **确认不再需要**:
   - 卸载是不可逆的
   - 没有回收站
   - 确保所有重要数据已备份

3. **关闭应用**:
   - 关闭 KOOK 消息转发系统
   - 结束相关进程（任务管理器）

4. **管理员权限**:
   - 运行脚本需要管理员权限
   - 右键 → "以管理员身份运行"

---

### 卸载不会删除

- ✅ Python 环境
- ✅ Node.js 环境
- ✅ Git 工具
- ✅ Chrome 浏览器
- ✅ 其他程序的数据

---

### 重新安装

完全卸载后，如需重新安装：

```bash
# 1. 访问 GitHub
https://github.com/gfchfjh/CSBJJWT

# 2. 下载最新版本
# 从 Release 页面下载

# 3. 按照安装指南操作
# 查看 QUICK_START_WINDOWS.md
```
