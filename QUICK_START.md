## ❓ 常见问题速查

### Q1: Cookie导入一直失败？

**解决方案（v6.0.0）**:

1. **使用Chrome扩展**（最推荐）
   - 成功率99%，仅需5秒
   - 自动格式化，不会出错

2. **查看详细错误提示**
   - v6.0.0提供详细的错误信息和建议
   - 按照提示修改Cookie格式

3. **支持的格式**
   - JSON数组（最常见）
   - JSON对象
   - Netscape格式
   - HTTP Cookie头
   - 等10+种格式

### Q2: 账号一直显示"离线"？

**检查清单**:
```

☐ 防火墙是否拦截？
```

**快速解决**:
```
1. 账号管理 → 选择账号 → "🔄 重新登录"
2. 或重新导出Cookie
3. 查看日志中的详细错误信息
```

☐ 频道映射是否设置？
